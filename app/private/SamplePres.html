Czym jest Meteor?

Written in JavaScript on the Node.js platform, Meteor is an open-source Isomorphic Development Ecosystem (IDevE) for efficiently and painlessly developing web and mobile applications. The Meteor ecosystem seamlessly integrates all the components (tools, libraries, databases, and frameworks [frontend and backend]) necessary for building and deploying applications.

http://javascriptissexy.com/learn-meteor-js-properly/

Główne zalety

  • Izomorficzność

ten sam język dla back-endu, front-endu

i aplikacji mobilnych

z wykorzystaniem tych samych bibliotek

  • wszechstronny i zintegrowany ekosystem

na który składają się frameworki,

biblioteki, bazy danych i narzędzia

  • łatwość budowania i wdrażania (build and deploy)

								
									# instalacja
									$ curl https://install.meteor.com | sh

									# tworzenie projektu
									meteor create myapp

									# uruchomienie aplikacji lokalnie (z auto odświeżaniem i śledzeniem zmian)
									meteor

									# dodawanie zależności
									meteor add bootstrap
									meteor add less

									# wdrażanie
									meteor deploy foo.meteor.com
